## PubSub
* 一个库，在任何框架都能使用
* 作用：实现跨组件数据通信
* 在React中：兄弟组件、爷孙组件通信才会使用
* 使用：
  * 订阅 PubSub.subscribe('消息名称', callback)
    * 订阅只能做一次
    * 订阅方是消息的接收方
    * 订阅要在发布之前
  * 发布 PubSub.publish('消息名称', 数据)
    * 发布可能有多次
    * 发布方时发布消息方
* 原理：自定义事件  new Event() 
  * 原生js事件
    * 触发：用户做一些操作（点击按钮、移动鼠标、按下键盘。。。）
  * 自定义事件
    * 触发：手动触发，执行了特定函数从而触发的（publish）
